  1. 1. 容器互联

本文最后更新于 天前，文中部分描述可能已经过时。

容器互联

  • dao_testlaravel_3 即为所需要的互联后的容器/应用名称
  • laravel-mysql 即为需要链接的 mysql 容器名称
  • --link 链接
  • -d 容器后台运行
  • -p 80 暴露 dao_testlaravel_3 容器的 80 端口到宿主机的动态端口
  • daocloud.io/layoute/laravle-english 为所需拉取的镜像
