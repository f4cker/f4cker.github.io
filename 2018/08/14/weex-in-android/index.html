2. JSBundle 资源生成

Android 加载 weex 和加载 React Native 类似也是要提供个 JSBundle 文件. 我们 weex create 后生成的项目文件都是以 .vue 后缀名结尾的,在引入 Android 之前,我们需要进行些处理才能放入 Android 项目的 assets 目录中供 WeexSdk 调用: 在项目根目录运行 npm run build 编译完成后会在dist目录下生成文件,每个vue文件会生成对应的js, 然后我们将 dist 文件夹内的 *.js 文件拷贝到 Android Assets 目录下即可
